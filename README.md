## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/roalexandru/PMagents.git
   cd PMagents
   ```
2. Open the chosen agent folder in UiPath Studio.  
3. Install required packages (listed in each agent folder).  
4. Run the workflow.
